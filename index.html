Inclusive Internationalization - VR Autism Therapy Tool

This project involves the development of a VR tool designed to assist in autism therapy. It provides therapists with customizable virtual environments for conducting stress exposure exercises.

Key Features:

  • Customizable virtual environments for various scenarios
  • Adjustable stress levels and stimuli
  • Real-time monitoring and data collection for therapists
  • Gradual exposure techniques for anxiety management

Technical Details:

  • Development Platform: Unity
  • VR Integration: Compatible with major VR headsets
  • Scripting: C# for game logic and interactions
  • Data Management: Secure storage and analysis of session data

Challenges Overcome:

A significant challenge was creating environments that were both engaging and not overstimulating for users with autism. We worked closely with autism specialists to fine-tune the sensory aspects of each virtual scenario.

Impact:

This tool has the potential to revolutionize autism therapy by providing a safe, controlled environment for exposure therapy. It allows for personalized treatment plans and can be easily adapted to individual needs.

BlackJack - Browser-based Card Game

This project is a browser-based implementation of the classic casino game, Blackjack, using only HTML, CSS, and JavaScript.

Key Features:

  • Fully functional Blackjack game logic
  • Interactive user interface for gameplay
  • Score tracking and win/loss statistics
  • Responsive design for various screen sizes

Technical Details:

  • HTML5 for structure
  • CSS3 for styling and animations
  • Vanilla JavaScript for game logic and DOM manipulation
  • LocalStorage for persisting game stats

Challenges Overcome:

A key challenge was implementing the complex rules of Blackjack, including handling Aces as 1 or 11. We solved this by creating a flexible scoring system that automatically adjusts the value of Aces based on the current hand.

Learning Outcomes:

  • Strengthened core web development skills
  • Improved understanding of game logic implementation
  • Enhanced skills in creating interactive web applications without frameworks
